Features

Everything you need, on a single live canvas

No tab-switching, no recording-then-reviewing. Parley surfaces insight while the conversation is still happening.

Dual-source capture

Captures your microphone and system audio at the same time, so every line is tagged as me or them.

Realtime transcription

Soniox streaming transcription with low latency, speaker diarization, and editable speaker names.

Switchable AI provider

Run analysis through Anthropic Claude or OpenRouter — pick whichever you prefer in Settings.

Live Q&A sidebar

Ask anything about the ongoing conversation and get streamed answers grounded in the live transcript.

Configurable evaluations

Preset and custom evaluation cards surface insights as the conversation unfolds — automatic or manual rerun.

TODO with AI auto-check

Track what you want to cover; items check off automatically as the AI detects they've been addressed.

Traditional Chinese

On-the-fly conversion of transcribed text to Traditional Chinese, right in the transcript.

Native, focused chrome

A clean custom titlebar and native-feeling window — designed to sit quietly beside your call.

How it works

From audio to insight in three steps

  1. 01

    Capture both sides

    Parley taps your system audio via Core Audio and your mic via cpal — no virtual cables, no setup.

  2. 02

    Transcribe live

    Two Soniox sessions stream transcription with diarization, labeling each line by speaker.

  3. 03

    Evaluate & answer

    AI evaluations and the Q&A sidebar run continuously, surfacing what's missing and what matters.

Get started

Run Parley in two commands

Clone the repo, add your API keys in Settings, and you're listening. macOS only for now — the audio layer sits behind a trait, so other platforms are possible.

# from the repo root
bun install
bun run tauri dev
View on GitHub Read the setup guide

Requires a Soniox key and an Anthropic or OpenRouter key — entered in-app, no env setup needed.
